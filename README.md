# موقع شركة العدل (Al Adl Company) 🚀
### الحلول المتكاملة لماكينات التصوير والطابعات ريكو (Ricoh)

هذا المستودع يحتوي على الكود المصدري لموقع **شركة العدل**، وهو موقع تعريفي وتفاعلي مصمم لخدمة عملاء الشركة في مجال بيع وصيانة ماكينات التصوير ريكو في مصر.

---

## 🛠 التقنيات المستخدمة (Tech Stack)
الموقع مبني باستخدام تقنيات الويب الأساسية لضمان السرعة القصوى والتوافق مع محركات البحث:
- **HTML5**: لهيكلة المحتوى.
- **Tailwind CSS**: لتصميم واجهة عصرية وسريعة الاستجابة (Responsive).
- **Vanilla JavaScript**: لإدارة منطق الموقع، نظام البحث، والتنقل بين الصفحات (SPA).
- **Font Awesome**: للأيقونات التفاعلية.
- **Google Fonts**: لاستخدام خطوط عربية احترافية (Tajawal, Cairo, Almarai).

---

## ✨ المميزات الرئيسية
1. **تصفح المنتجات**: نظام عرض متطور لماكينات التصوير (أبيض وأسود / ألوان) مع مواصفات تقنية دقيقة.
2. **بورصة الورق اليومية**: لوحة بيانات تفاعلية تعرض أسعار أنواع الورق المختلفة في السوق المصري.
3. **نظام طلب الصيانة**: نماذج ذكية لتبليغ الأعطال، طلب قطع الغيار، والشكاوى، مربوطة مباشرة بـ WhatsApp.
4. **دعم فني متكامل**: عرض أنواع عقود الصيانة (سنوية، شاملة، زيارة حرّة).
5. **تحسين محركات البحث (SEO)**: الموقع مهيأ بالكامل بكلمات مفتاحية ووسوم Meta Tags لضمان تصدر نتائج البحث.
6. **تجربة مستخدم سلسة**: نظام تنقل داخلي دون إعادة تحميل الصفحة (Single Page Application feel).

---

## 📂 هيكل المشروع
- `index.html`: الملف الرئيسي الذي يحتوي على هيكل الموقع بالكامل والسكربتات.
- `assets/`: (اختياري) المجلد الذي يحتوي على الصور واللوجو الخاص بالشركة.
- `scripts`: يحتوي ملف الـ HTML على منطق برمجي لإدارة قاعدة بيانات المنتجات (`productsDB`) والأخبار (`newsDB`) محلياً.

---

## 🚀 كيفية التشغيل
الموقع لا يحتاج إلى سيرفر خاص أو قواعد بيانات معقدة، يمكنك تشغيله ببساطة عبر:
1. تحميل الملف.
2. فتح ملف `index.html` في أي متصفح.
3. أو استضافته مجاناً عبر **GitHub Pages**.
